# Calculator

A basic calculator app, with keyboard support for numbers, dot and backspace. 

This repository is built in the context of The Odin Project foundations course: https://www.theodinproject.com/lessons/foundations-calculator

You can try out the calculator here: https://gregorgatej.github.io/calculator/
